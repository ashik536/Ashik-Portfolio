# Ashik Portfolio

This repository contains the static portfolio site for Ashik.

## Overview

A simple, single-page portfolio website that showcases projects, skills, and contact information.

## Files

- `Ashik-Portfolio.html` — main portfolio page (open in a browser to view).

## How to view

1. Open `Ashik-Portfolio.html` in your web browser.
2. (Optional) Use the VS Code Live Server extension to serve and preview locally.

## Customize

Replace placeholder text (about, projects, contact) in `Ashik-Portfolio.html` with your details.

## Contact

Add your contact details or links (email, LinkedIn, GitHub) to the HTML page.
